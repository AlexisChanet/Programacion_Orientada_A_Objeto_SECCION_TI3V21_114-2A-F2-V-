# Evaluación 1 - Programación Orientada a Objetos

Autor: Alexis Castillo  
Sección: TI3V21114-2A-F2V  
Repositorio oficial para los ejercicios de POO

Este repositorio contiene los ejercicios evaluados en la asignatura **Programación Orientada a Objetos**, desarrollados en Python.

Cada carpeta (ejercicio1 a ejercicio7) contiene:
- Un archivo main.py con ejemplos de ejecución.
- Una carpeta clases con las clases y métodos implementados.
- Archivos __init__.py para definir los paquetes.

Todos los programas cumplen con los criterios funcionales solicitados en la evaluación.
